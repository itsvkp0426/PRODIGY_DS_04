### Objective
To analyze and visualize sentiment patterns in social media data in order to understand public opinion and attitudes towards different topics or brands.

---

### Dataset
Source: Twitter Entity Sentiment Dataset

Description:
The dataset contains tweets related to various brands and topics along with their associated sentiment labels such as positive, negative, neutral, and irrelevant.

---

### Data Preparation
- Loaded the dataset using pandas.
- Assigned appropriate column names to the dataset.
- Removed records with missing tweet text.

---

### Analysis and Visualizations
The following analyses were performed:
- Visualization of overall sentiment distribution using a bar chart.
- Identification of the top 5 most discussed entities.
- Visualization of sentiment patterns across the top 5 entities using grouped bar charts.

---

### Observations
- Negative and neutral sentiments are more frequent compared to positive sentiment.
- Sentiment distribution varies across different entities, indicating differing public opinions.

---

### Conclusion
This analysis provides a basic understanding of sentiment trends in Twitter data and demonstrates how visualization techniques can be used to interpret public sentiment towards specific topics or brands.
